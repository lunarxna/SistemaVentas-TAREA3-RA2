# SistemaVentas-TAREA3
Ejercicio sobre la creación de un Sistema de Ventas

Descripción: 
Este es un programa realizado con la programación en capas utilizando el lenguaje C#, el cual permite la gestión de ventas permitiendo agregar nuevas ventas y al final se genera la factura de la misma; permite la gestión de productos con las opciones de editar los que ya están, agregar nuevos o eliminarlos, y así con los demás formularios de Categorías, Clientes y Facturación. Permitiendo así la correcta gestión de un sistema de ventas.

Herramientas utilizadas:
- Lenguaje: C#
- Framework: .NET Framework
- IDE: Visual Studio 2022
